beforeUpdate

abstract fun beforeUpdate(model: M, event: E): Unit

Called right before the Update.update function is called.

This method mustn't block, as it'll hinder the loop from running. It will be called on the same thread as the update function.

Parameters

model - the model that will be passed to the update function

event - the event that will be passed to the update function
