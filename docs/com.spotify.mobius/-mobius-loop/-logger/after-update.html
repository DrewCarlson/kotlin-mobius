afterUpdate

abstract fun afterUpdate(model: M, event: E, result: Next<M, F>): Unit

Called right after the Update.update function is called.

This method mustn't block, as it'll hinder the loop from running. It will be called on the same thread as the update function.

Parameters

model - the model that was passed to update

event - the event that was passed to update

result - the Next that update returned
