fun observe(observer: Consumer<M>): Disposable

Add an observer of model changes to this loop. If .getMostRecentModel is non-null, the observer will immediately be notified of the most recent model. The observer will be notified of future changes to the model until the loop or the returned Disposable is disposed.

Parameters

observer - a non-null observer of model changes

Exceptions

NullPointerException - if the observer is null

IllegalStateException - if the loop has been disposed

Return
a Disposable that can be used to stop further notifications to the observer
